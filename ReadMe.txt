1.Folder  "Original_Exercises" stores the screenshots of the original exercises.
2.Folder "New_Exercises" stores the newly generated exercises. Open the "index. html" under each subfolder in the browser to view the details of the exercises.
3. Meaning of columns in "Evaluation_by_expert.csv"：
    Eid: The id of original exercises.
    id: The id of newly generated exercises.
    value_1 and value_2 ：Expert evaluation results. "0" means the exercise be used in teaching, while "1" means the exercise  cannot be used in teaching.
